# Instructions for Agents

## What this application does

It is an example starter repository for feature applications.
It contains authentication, mailing, database, a frontend and a server.

## How is the server structured

The server source code is under "server" directory.

## Shared directory

The "shared" directory is where the database schema is defined. The types are used both
by frontend and server.

## Rules

- Use `pnpm dlx` instead of `npx` for commands.
- The code should explain itself. To many comments decreases readability.
